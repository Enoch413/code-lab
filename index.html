시험 운영 센터

SUPERADMIN ONLY
현재 시험 정보를 불러오는 중입니다.
최고 관리자만 시험 종료 아카이브, 반/학생 정리, 새 시험 초기화를 진행할 수 있습니다.
STEP 1

시험 종료 아카이브

현재 CODE LAB 운영 데이터 전체를 엑셀과 JSON 스냅샷으로 저장합니다.

아직 아카이브를 실행하지 않았습니다.
STEP 2

반 / 관리자 재배정

현재 반 목록과 각 반 담당 관리자 배정을 CSV로 내려받아 수정한 뒤 다시 업로드합니다.

아직 업로드된 반/관리자 변경안이 없습니다.
STEP 3

학생 변동 반영

신규 학생 추가, 기존 학생 수정, 탈퇴 학생 disabled 처리를 CSV로 일괄 반영합니다.

아직 업로드된 학생 변경안이 없습니다.
STEP 4

새 시험 시작 초기화

반 / 학생 / 문제 라이브러리를 제외한 시험 운영 데이터만 비우고 새 시험 key를 발급합니다.

초기화는 아카이브와 반/학생 변경 반영 후 실행하는 것을 권장합니다.
